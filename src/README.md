# Interfaces em Java ☕

Um projeto focado na aplicação prática e na compreensão do conceito de **Interfaces** na linguagem Java.

## 🚀 Sobre o Projeto

Este repositório contém implementações criadas do zero para demonstrar como as interfaces funcionam na prática. O objetivo principal é consolidar conhecimentos sobre Programação Orientada a Objetos (POO), estabelecimento de contratos entre classes, polimorfismo e boas práticas de desenvolvimento.

## 🛠️ Tecnologias Utilizadas

* **Java** 
* **Git** e **GitHub**

## ⚙️ Como Executar na Sua Máquina

Para clonar e testar este projeto localmente, você precisará ter o [Git](https://git-scm.com) e o [Java JDK](https://www.oracle.com/java/technologies/downloads/) instalados no seu computador.

No seu terminal, digite os seguintes comandos:

```bash
# Clonar o repositório
git clone [https://github.com/iamfrc/interfaces1-java.git](https://github.com/iamfrc/interfaces1-java.git)

# Acessar a pasta do projeto
cd interfaces1-java

# A partir daqui, você pode abrir o projeto na sua IDE preferida (como IntelliJ IDEA, Eclipse ou VS Code) para compilar e executar o código.
